# Morpheus

A cross-platform 2D game built with SDL2 and C++.

## Prerequisites

### macOS
```bash
brew install sdl3
```

### Ubuntu/Debian
```bash
sudo apt-get install libsdl3-dev
```

### Windows
Download SDL3 development libraries from [libsdl.org](https://www.libsdl.org/download-2.0.php)

## Building

```bash
cd build
cmake ..
cmake --build .
```

## Running

```bash
./Morpheus
```

## Controls

- **ESC** - Exit game
